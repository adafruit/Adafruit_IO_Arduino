# Adafruit IO Arduino Library
[![Documentation Status](https://readthedocs.org/projects/adafruit-io-arduino/badge/?version=latest)](adafruit-io-arduino.readthedocs.io) [![Build Status](https://travis-ci.org/adafruit/Adafruit_IO_Arduino.svg?branch=master)](https://travis-ci.org/adafruit/Adafruit_IO_Arduino) [![Discord](https://img.shields.io/discord/327254708534116352.svg)](https://discord.gg/nBQh6qu)


This library provides a simple device independent interface for interacting with [Adafruit IO](https://io.adafruit.com) using Arduino.
It allows you to switch beween WiFi (ESP8266, M0 WINC1500, & WICED), Cellular (32u4 FONA), and Ethernet (Ethernet FeatherWing)
with only a two line change in your sketch.

## Documentation and Usage
Documentation including installation, configuration and usage is found on [the ReadTheDocs page for this repository](adafruit-io-arduino.readthedocs.io).
